---

# Mobile UX Principles

Good mobile design should:

- reduce cognitive load
- prioritize the most important information
- make actions easy to reach
- minimize navigation depth

Primary actions should be reachable within **thumb-friendly zones**.

---

# Tablet Adaptation

Explain how the mobile interface scales to tablet screens.

Possible changes:

- multi-column layouts
- sidebar navigation
- expanded dashboards

Tablet interfaces can show **more simultaneous information**.
