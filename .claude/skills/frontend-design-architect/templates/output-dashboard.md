# Style DNA

Define the visual identity of the dashboard.

Typography:

- strong headline hierarchy
- readable body text
- compact UI-friendly fonts

Colors:

- neutral background palette
- accent color for highlights
- clear status colors (success, warning, error)

Spacing:

- consistent spacing rhythm
- compact but readable density

Shape Language:

- card corners
- component geometry

Elevation:

- subtle card shadows
- layered surfaces

Dashboards should feel **organized, structured, and calm**.
